# Dash HD Util

Util / Tests for BIP32 wallets in Dash using Bitcore-lib-dash

## Download / Install / Run

    git clone https://github.com/andyfreer/dash-hd-util.git
    cd dash-hd-util/
    npm install

Run some test cases..

    npm start

## Testing
Test using Mocha

    npm install -g mocha

    npm test
